# Trend-DiffNet

Before training, please configure the following paths in `Train_TDNet_Charades_CLIP.py`:

- `rgb_root` should point to the extracted feature directory generated by pretrained models (e.g., I3D or CLIP features).

- `annotation_file` should point to:
  
```text
data/charades.json
```

Example:

```bash
python Train_TDNet_Charades_CLIP.py \
    -rgb_root /path/to/CLIP_or_I3D_features \
    -annotation_file data/charades.json
```
